Definition and Usage

The <html> tag represents the root of the HTML document.

The <html> tag is the container for all other HTML elements (except for the <!DOCTYPE> tag)




Note: You should always include the langattribute inside the <html> tag, to declare the language of the Web Page. This is meant to assist search engines and browsers.

GlobalAttributes: The <html> tag also supports the Global Atrributes in HTML.

Syntax: <html>....</html>
